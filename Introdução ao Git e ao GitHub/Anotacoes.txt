SINCRONIZANDO GIT (LOCAL) COM O GITHUB (NUVEM):

1 - Abrir Arquivo txt em diretório

2 - Git: ls (listar) para verificar se o arquivo existe

3 - Git entender que a gente quer enviar para o controle de versão/Local: 
git add . ou git -A (pasta toda)

4 - Git entender que quer salvar apenas um arquivo:
git add "nome do arquivo"

5 - Git entender que a gente quer enviar para o controle de versão/inclusão no git/github: 
git commit -m "Inclusão das anotações do curso do Git/GitHub"

6 - Git Push:
git push
Sincroniza repositório local com o repositório GitHub
------------------------------------------------------------------------------------------------
REMOVER ARQUIVO GIT: 
rm <nome do arquivo>

REMOVER ARQUIVO DO GITHUB:
git commit -m "Exclusão das anotações do curso do Git/GitHub"
