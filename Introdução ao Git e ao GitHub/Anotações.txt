Tabela de "Atalho":

ls -a: Ver arquivos ocultos
cd: mudar diretório/pasta
cd ..: sair do diretório/pasta
tab: auto completar
Abrir Diretório: dir
Abrir/Criar Pasta/diretório: mkdir "nome da pasta"

1 - Git entender que a gente quer enviar para o controle de versão/Local: 
git add. ou git -A 

2 - Git entender que a gente quer enviar para o controle de versão/inclusão no git/github: 
git commit -m "Inclusão das anotações do curso do Git/GitHub"


Untracked files:

tracked files: (adicionar)
git add .
